# Product Overview

## Project Purpose
Macca is an AI-powered English speaking coach specifically designed for Indonesian learners. The platform focuses on speaking and listening practice with real-time, personalized feedback to help users improve their English communication skills.

## Value Proposition
- **Voice-First Learning**: Microphone-based interaction as the primary interface for natural conversation practice
- **Real-Time Feedback**: Immediate, structured feedback on grammar, vocabulary, and pronunciation
- **Personalized Experience**: AI-driven coaching that adapts to individual learner needs
- **Indonesian Context**: Tailored for Indonesian English learners with bilingual support

## Key Features

### Learning Modes
1. **Live Conversation** - Free-form conversation practice with structured feedback after each turn
2. **Guided Lessons** - Structured learning tasks with progressive difficulty and lesson tracking
3. **Pronunciation Coach** - Focused practice on specific sounds and pronunciation patterns

### Core Capabilities
- **Speech Recognition (ASR)**: Converts spoken English to text using Hugging Face Whisper models
- **AI Conversation**: Natural language interaction powered by SeaLLMs language models
- **Text-to-Speech (TTS)**: Audio responses for listening practice using Seamless M4T models
- **Structured Feedback System**: 
  - Grammar corrections with explanations
  - Vocabulary suggestions and alternatives
  - Pronunciation analysis and tips
  - Bilingual explanations (English/Indonesian)
- **Session Management**: Track learning progress across multiple conversation sessions
- **Vocabulary Tracking**: Save and review new words and phrases
- **Audio Storage**: Record and replay conversation audio for review

### Development Features
- **Mock AI Mode**: Full development environment without external API dependencies
- **JWT Authentication**: Secure user signup and login system
- **RESTful API**: Clean API design for frontend-backend communication
- **Database Persistence**: PostgreSQL storage for users, sessions, and learning data

## Target Users
- Indonesian students learning English
- Professionals improving business English communication
- Anyone seeking conversational English practice with AI coaching
- Language learners preferring voice-based interaction over text

## Use Cases
- Daily English conversation practice
- Pronunciation improvement for specific sounds
- Vocabulary building through contextual learning
- Preparation for English speaking tests or interviews
- Confidence building in spoken English communication
